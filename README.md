# Welcome to the Temperature Data Visualization Project! 📊

This repository demonstrates how to turn clean time-series data into clear statistical graphs using **Matplotlib** and **Seaborn**.

While Pandas cleans the logs, this script sets up a two-panel visual dashboard to display trends and data distribution shapes without any overlapping labels.

---

## 🛠️ The Visual Engine: What the Code Does

This script reads your clean data and draws the graphs using three structured steps:

* **Dual Plot Setup:** It creates a single display window with two separate chart spaces stacked vertically, automatically adjusting the padding so headings and labels do not overlap.
* **Trend Line Tracking:** It maps the clean dates chronologically and draws a smooth royal blue line across the timeline to highlight warm periods, cool drops, and cycles over the year.
* **Distribution Grouping:** It groups temperatures into vertical coral bars to show which temperature ranges happened most often, then overlays a smooth trend wave line on top.
